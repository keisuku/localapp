## 1. このアプリは何をするもの？

Excel や CSV のファイルを画面にドラッグ&ドロップするだけで、

- 一覧表・カード・カンバン（付箋ボード）・ダッシュボードで**見やすく管理**
- キーワード検索・絞り込み・並べ替え・タグ付け
- 自動の**点数付け**（例: 求人なら年収や働きやすさから総合スコアを計算）
- Excel / CSV / JSON への**書き出し**

ができる、**パソコンのブラウザだけで動く**業務アプリです。
「求人比較」「保険案件管理」「汎用データ管理」の 3 つが最初から入っており、他の業務（在庫管理など）にも部品を足すだけで広げられる作りになっています。

## 2. いちばん大事なこと：データはどこに保存される？

```
┌─────────── あなたのパソコン ───────────┐
│                                                │
│   ブラウザ（Chrome / Edge など）               │
│   ┌────────────────────────┐   │
│   │  Local App Studio（このアプリ）        │   │
│   │      ↓ 保存          ↑ 読み込み       │   │
│   │  ブラウザ内の保存領域（IndexedDB）     │   │
│   └────────────────────────┘   │
│                                                │
└────────────────────────────────┘
            ✕ 外部のサーバーには何も送りません
```

- 取り込んだデータ・入力した内容は、**すべてあなたのパソコンのブラウザの中**に保存されます。
- **インターネット上のサーバーには一切送信されません。** クラウドにも上がりません。
- そのため、ネットを切った状態でも全機能が動きます。

### よくある質問

| 質問 | 答え |
|---|---|
| データはクラウドに上がりますか？ | いいえ。パソコンの外には出ません。 |
| 会社の顧客データを入れても外部に漏れない？ | アプリ自体は外部送信しません。ただし PC 自体の管理（紛失・のぞき見・マルウェア）は別問題なので、会社のルールに従ってください。 |
| ネットがない場所でも使えますか？ | 使えます。「オフライン版（1 ファイル）」をダウンロードすれば、完全にネット無しの PC でも動きます。 |
| 他の人と同じデータを見られますか？ | できません（共有機能はありません）。共有したい場合はバックアップファイルや Excel 出力を渡してください。 |
| スマホでも使えますか？ | 画面は PC 用に最適化しています。PC でのご利用を前提にしてください。 |

## 3. 知っておくべきリスクと対策

| リスク | 何が起きるか | 対策 |
|---|---|---|
| **ブラウザのデータ削除** | ブラウザの「閲覧履歴データの削除」をすると、アプリのデータも一緒に消えます | 設定画面の「バックアップを保存」で JSON ファイルを定期的に保存。ファイルを画面に落とすだけで復元できます |
| **パソコンの故障・紛失** | PC が壊れる/失くすとデータも失われます。紛失時は第三者に見られる可能性もあります | バックアップを別の場所（社内共有フォルダ等）に保管。PC のログインパスワードとディスク暗号化（BitLocker 等）を有効に |
| **共有 PC での利用** | 同じ PC・同じブラウザを使う人はデータを見られます | 共有 PC では使わない。使う場合は OS のアカウントを分ける |
| **間違った操作** | 誤って削除・上書きする可能性 | 削除や初期化の前には必ず確認画面が出ます。実行後も「元に戻す」ボタンで復元できます（直近 50 操作） |
| **古いファイルの放置** | 出力した Excel やバックアップを放置すると情報漏えいの元に | 不要になった出力ファイルは削除する運用に |

## 4. 機密データを扱う場合のおすすめ運用

1. 会社の情報管理ルール上、そのデータを個人 PC で扱ってよいか先に確認する。
2. PC のディスク暗号化と画面ロックを必ず有効にする。
3. 週 1 回などタイミングを決めて JSON バックアップを取り、決められた保管場所に置く。
4. 配布する場合は「オフライン版 HTML」を IT 部門経由で配ると安全です（中身の検証ができるため）。

## 5. 提供形態

| 形態 | 説明 | 向いている場面 |
|---|---|---|
| GitHub Pages 版 | URL を開くだけで最新版が使える | 普段使い |
| オフライン版（単一 HTML） | 1 ファイルをダブルクリックで起動。ネット不要 | ネット分離環境・社内配布 |
| アプリとしてインストール（PWA） | ブラウザの「インストール」でデスクトップアプリ風に | 毎日使う人 |

困ったときは、まず設定画面の「データの保存について」と、この文書の表を確認してください。
