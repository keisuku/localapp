# 会社のChatGPTで「センス良く」再現するレシピ（暗記版）

> 目的：今回作った **予実・差分比較ツール**（2つのExcelを比べて差分だけ出す）の類を、
> 会社のChatGPTでサッと再現する。**暗記できる最小限**だけ載せる。

---

## 覚えるのはこれだけ：「1・3・3」＋「白紺・余白・角丸・影うすめ」

- **1** … 成果物は **HTML 1ファイル**（保存してブラウザで開けば動く）
- **3** … 技術は **3つだけ**＝ **素のHTML/JS ＋ Tailwind(CDN) ＋ SheetJS(CDN)**
- **3** … 機能は **3ステップ**＝ **①入れる → ②選ぶ → ③Excelで出す**
- **白紺・余白・角丸・影うすめ** … 見た目の合言葉（白背景×ネイビー、余白広め、角を丸く、影は控えめ＝shadcn/ui風）

> これだけ覚えれば、プロンプトはその場で組み立てられます。

---

## マスタープロンプト（穴埋め。構造だけ覚える）

```
あなたは丁寧なフロントエンドエンジニアです。
「____（ツール名）____」を HTML 1ファイルだけで作って。

・技術：素のHTML/JS ＋ Tailwind(CDN) ＋ SheetJS(CDN)。サーバー不要・データは外に出さない。
・手順は3ステップ：① ____入れる____ ② ____選ぶ____ ③ ____Excelで出す____
・見た目：白×ネイビー、余白広め、角丸、影うすめ（shadcn/ui風）。1画面・3ステップで迷わせない。
　ドラッグ&ドロップ対応と「サンプルで試す」ボタンも付けて。
・仕上げ：コピペで保存→ブラウザで開けば動く完全な1ファイルで出力。最後に「外部送信していない」ことを一言で。
```

### 今回の予実・差分ツール用の埋め方（例）

- ツール名：**予実・差分比較**
- ① **2つのExcel（前回/今回・予算/実績）を入れる**
- ② **キー列（顧客名）・区分列・金額列を選ぶ**
- ③ **差分（新規/削除/変更）だけをExcelで出す**

### ★そのまま貼れる完成版（予実だけ作りたい人はこれをコピペ）

```
あなたは丁寧なフロントエンドエンジニアです。
「予実・差分比較ツール」を HTML 1ファイルだけで作って。

・技術：素のHTML/JS ＋ Tailwind(CDN) ＋ SheetJS(CDN)。サーバー不要・データは外に出さない。
・やること（3ステップ）：
　① 2つのExcel/CSVを入れる（前回＝予算 / 今回＝実績）。ドラッグ&ドロップ対応。
　② キー列（例：顧客名）・区分列（任意・複数可）・金額列を選ぶ。
　③ 同じキー＋区分で金額を合計して突き合わせ、新規・削除・変更・同一に分類。
　　 差分（前回額／今回額／差額／差分率）を表で表示し、「差分だけ」をExcelで出力。
・見た目：白×ネイビー、余白広め、角丸、影うすめ（shadcn/ui風）。1画面・3ステップで迷わせない。
　「サンプルで試す」ボタン（ダミーの予算/実績）も付けて。
・仕上げ：コピペで保存→ブラウザで開けば動く完全な1ファイルで。最後に「外部送信していない」ことを一言で。
```

> 本番の顧客データは貼らず、「サンプルで試す」で動作確認 → 手元のファイルで使う。

---

## 手順（5ステップ）

1. 上のプロンプトを穴埋めして **ChatGPTに貼る**。
2. 出てきた **HTMLを丸ごとコピー** → `tool.html` で保存。
3. **ダブルクリックで開く**（ブラウザで動く）。
4. 直したい所だけ追撃：**「ここを〇〇に」「差分だけ赤字に」** など1つずつ。
5. 最後に **「外部通信していないか確認して」** と言わせて締める。

---

## 効く追撃プロンプト（3つだけ覚える）

- **「迷わせないで」**… 「次に何をすればいいか」をステップ表示・ボタンで明示させる。
- **「センス良く」**… 余白・行間・色のコントラストを整えさせる（白紺・余白・角丸・影うすめ、と添える）。
- **「1ファイルのまま」**… 機能追加しても**HTML1枚を崩さない**よう毎回釘を刺す。

---

## ワンランク上（任意・覚えなくてよい）

- **完全オフライン**にしたい：「SheetJSもCDNでなく**インライン**にして、ネット不要の1ファイルに」と頼む。
  （ファイルは大きくなるが、開く時もネット不要になる）
- **きれいな部品**が欲しい：「shadcn/ui 風の **Button / Card / Tabs** を自前の小さなCSSで」と頼む。
- **本格版**（今回のリポジトリ相当）にしたい時の技術一式：
  React + TypeScript + Vite + Tailwind + shadcn/ui + Zustand + Dexie(IndexedDB) + SheetJS。
  ※ これは1ファイルでは収まらないので「プロジェクト一式」を順に出させる。

---

## 注意（会社利用の前提）

- ChatGPTに**実データ（顧客名・金額）は貼らない**。サンプルのダミーで作らせ、本番データは手元のファイルで動かす。
- SheetJSは脆弱性修正のため **0.20系（最新）** を使う（CDN例：`https://cdn.sheetjs.com/xlsx-0.20.3/package/dist/xlsx.full.min.js`）。
- 配る前に **IT/セキュリティの確認**を。「外部通信ゼロ」は強い説明材料。
